# Part Numbering
Numbering scheme: 

  PPPP-pp-rr

Where:
 - PPPP: Project number
 - pp: Part number(eg. pcb, casing, stencil are parts)
 - rr: Revision of part

## Sensof HT v1
Project number: 0002

Parts:
00 - Sensof HT v1 PCB
01 - Top layer solder stencil for Sensof HT v1 PCB

## Sensof HT v2
Project number: 0003

Parts:
00 - Sensof HT v2 PCB
01 - Top layer solder stencil for Sensof HT v2 PCB
02 - Bottom layer solder stencil for Sensof HT v2 PCB
